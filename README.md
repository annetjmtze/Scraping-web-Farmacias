# 🤖 Bot de Telegram - Buscador de Medicamentos

Este es un bot asistente de Telegram desarrollado con Python y la biblioteca `python-telegram-bot`, integrado con la API de Anthropic (Claude Haiku 4.5) para consultar información sobre medicamentos en México.

### 🚀 Probar el Bot
Puedes interactuar con el bot directamente aquí: @Dr_ahorro_bot

### 🛠️ Características y Comandos
* `/start`: Saludo inicial.
* `/ayuda`: Instrucciones de uso.
* `/medicamento <nombre>`: Devuelve una ficha técnica limpia extraída mediante IA en formato JSON.
* **Función Echo**: Cualquier otro mensaje de texto será repetido por el bot.
