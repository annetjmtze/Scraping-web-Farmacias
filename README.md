# 🤖 Bot de Telegram - Buscador de Medicamentos en México

Este es un bot asistente de Telegram desarrollado en Python que utiliza la biblioteca `python-telegram-bot` e integra la API de Anthropic (Claude Haiku) para consultar fichas técnicas de medicamentos de manera estructurada.

### 🚀 Link del Bot Activo en Telegram
Puedes interactuar con el bot directamente aquí: [@Dr_ahorro_bot](https://t.me/Dr_ahorro_bot)

---

## 💻 Configuración y Ejecución Local

### Prerrequisitos
* **Python 3.10 o superior** (Desarrollado y probado en Python 3.13)
* Una cuenta de Telegram y un Token de bot (obtenido mediante `@BotFather`)
* Una API Key válida de Anthropic
